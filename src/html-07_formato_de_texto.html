Formato de Texto HTML

Introducción

En el fascinante mundo del desarrollo web, dar estilo y estructura al texto es fundamental para crear páginas atractivas y fáciles de leer. Hoy, se invita a los lectores a sumergirse en el poder de HTML, el lenguaje esencial para la web. Se descubrirán las etiquetas clave que permiten formatear el texto. Desde negritas y cursivas hasta el tamaño y la organización, el objetivo es transformar las palabras en contenido visualmente impactante.

  • <mark> (Sombrero amarillo): Imaginarse que el sombrero amarillo es algo que se quiere resaltar o marcar como importante, como si se subrayara con un marcador fluorescente. En HTML, la etiqueta <mark> se usa para resaltar texto con un color de fondo, como un marcador. Es útil para llamar la atención sobre una parte específica de una frase.
  • <em> (Boina marrón): La etiqueta <em> significa énfasis (de emphasis). Se usa para dar énfasis a una palabra o frase, generalmente mostrándola en cursiva. Es como cuando se cambia el tono de voz para enfatizar algo que se dice.
  • <strong> (Chaqueta amarilla del hombre): La chaqueta amarilla que se está probando el hombre es muy llamativa y destaca. La etiqueta <strong> se usa para indicar que un texto tiene gran importancia o relevancia, y por defecto, los navegadores lo muestran en negrita. Es como cuando usas mayúsculas o gritas una palabra para que quede claro que es lo más importante.
  • <del> (Camiseta naranja): La camiseta naranja ya no es útil o se quisiere desechar. La etiqueta <del> (de delete) se usa para indicar que un texto ha sido eliminado o ya no es válido, mostrándolo por defecto con una línea que lo tacha. Es útil cuando se quiere mostrar un cambio, por ejemplo, un precio anterior que ha sido reemplazado por uno nuevo.
  • <u> (Corbata): La corbata está ""subrayada"" por su posición. La etiqueta <u> (de underline) se usa para subrayar texto. Aunque en la web es menos común usarla para subrayar por sí sola (ya que a menudo el subrayado indica un enlace), todavía puede usarse para propósitos estilísticos donde el significado de subrayado es relevante.
